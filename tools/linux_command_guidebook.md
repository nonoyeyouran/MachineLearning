## 文件与目录操作

### 1. ls
- **功能**：列出目录内容。
- **常用选项**：
  - `-l`：以长格式显示。
  - `-a`：显示所有文件（包括隐藏文件）。
  - `-h`：以人类可读的格式显示文件大小。
- **示例**：
  - `ls -l /home`

### 2. cd
- **功能**：切换当前工作目录。
- **示例**：
  - `cd /var/log`

### 3. pwd
- **功能**：显示当前工作目录的路径。
- **示例**：
  - `pwd`

### 4. mkdir
- **功能**：创建目录。
- **常用选项**：
  - `-p`：递归创建目录。
- **示例**：
  - `mkdir -p /home/user/newdir`

### 5. rm
- **功能**：删除文件或目录。
- **常用选项**：
  - `-r`：递归删除目录。
  - `-f`：强制删除，不提示确认。
- **示例**：
  - `rm -rf /home/user/olddir`

### 6. cp
- **功能**：复制文件或目录。
- **常用选项**：
  - `-r`：递归复制目录。
  - `-p`：保留文件属性。
- **示例**：
  - `cp -r /home/user/source /home/user/destination`

### 7. mv
- **功能**：移动或重命名文件或目录。
- **示例**：
  - `mv /home/user/oldname /home/user/newname`

### 8. touch
- **功能**：创建空文件或更新文件时间戳。
- **示例**：
  - `touch newfile.txt`

## 文件内容查看与编辑

### 1. cat
- **功能**：显示文件内容。
- **示例**：
  - `cat /etc/passwd`

### 2. more
- **功能**：分页显示文件内容。
- **示例**：
  - `more /var/log/syslog`

### 3. less
- **功能**：分页显示文件内容，支持上下滚动。
- **示例**：
  - `less /var/log/syslog`

### 4. head
- **功能**：显示文件开头部分。
- **常用选项**：
  - `-n`：指定显示的行数。
- **示例**：
  - `head -n 10 /var/log/syslog`

### 5. tail
- **功能**：显示文件末尾部分。
- **常用选项**：
  - `-n`：指定显示的行数。
  - `-f`：实时跟踪文件更新。
- **示例**：
  - `tail -f /var/log/syslog`

### 6. grep
- **功能**：在文件中搜索指定模式。
- **常用选项**：
  - `-i`：忽略大小写。
  - `-r`：递归搜索目录。
- **示例**：
  - `grep "error" /var/log/syslog`

### 7. nano
- **功能**：简单的文本编辑器。
- **示例**：
  - `nano /etc/hosts`

### 8. vim
- **功能**：强大的文本编辑器。
- **示例**：
  - `vim /etc/hosts`

## 系统信息与管理

### 1. uname
- **功能**：显示系统信息。
- **常用选项**：
  - `-a`：显示所有信息。
- **示例**：
  - `uname -a`

### 2. df
- **功能**：显示磁盘使用情况。
- **常用选项**：
  - `-h`：以人类可读的格式显示。
- **示例**：
  - `df -h`

### 3. du
- **功能**：显示目录或文件的磁盘使用情况。
- **常用选项**：
  - `-h`：以人类可读的格式显示。
  - `-s`：显示总大小。
- **示例**：
  - `du -sh /home`

### 4. top
- **功能**：实时显示系统进程和资源使用情况。
- **示例**：
  - `top`

### 5. ps
- **功能**：显示当前进程状态。
- **常用选项**：
  - `-ef`：显示所有进程。
- **示例**：
  - `ps -ef`

### 6. free
- **功能**：显示内存使用情况。
- **常用选项**：
  - `-h`：以人类可读的格式显示。
- **示例**：
  - `free -h`

## 用户与权限管理

### 1. useradd
- **功能**：创建新用户。
- **示例**：
  - `useradd newuser`

### 2. passwd
- **功能**：修改用户密码。
- **示例**：
  - `passwd newuser`

### 3. usermod
- **功能**：修改用户属性。
- **示例**：
  - `usermod -aG sudo newuser`

### 4. chmod
- **功能**：修改文件或目录权限。
- **示例**：
  - `chmod 755 script.sh`

### 5. chown
- **功能**：修改文件或目录的所有者。
- **示例**：
  - `chown user:group file.txt`

## 网络管理

### 1. ifconfig
- **功能**：显示和配置网络接口。
- **示例**：
  - `ifconfig eth0`

### 2. ping
- **功能**：测试网络连接。
- **示例**：
  - `ping google.com`

### 3. netstat
- **功能**：显示网络连接、路由表等信息。
- **常用选项**：
  - `-tuln`：显示所有监听端口。
- **示例**：
  - `netstat -tuln`

### 4. ssh
- **功能**：远程登录到另一台主机。
- **示例**：
  - `ssh user@remotehost`

### 5. scp
- **功能**：在本地和远程主机之间复制文件。
- **示例**：
  - `scp file.txt user@remotehost:/home/user`

## 进程管理

### 1. kill
- **功能**：终止进程。
- **常用选项**：
  - `-9`：强制终止进程。
- **示例**：
  - `kill -9 1234`

### 2. pkill
- **功能**：根据进程名终止进程。
- **示例**：
  - `pkill firefox`

### 3. bg/fg
- **功能**：将进程放到后台或前台运行。
- **示例**：
  - `bg %1`（将任务 1 放到后台）
  - `fg %1`（将任务 1 放到前台）

## 软件包管理

### 1. apt (Debian/Ubuntu)
- **功能**：管理软件包。
- **常用命令**：
  - `apt update`：更新软件包列表。
  - `apt install <package>`：安装软件包。
  - `apt remove <package>`：卸载软件包。
- **示例**：
  - `apt install vim`

### 2. yum (CentOS/RHEL)
- **功能**：管理软件包。
- **常用命令**：
  - `yum install <package>`：安装软件包。
  - `yum remove <package>`：卸载软件包。
- **示例**：
  - `yum install vim`

## 压缩与解压

### 1. tar
- **功能**：打包和解包文件。
- **常用选项**：
  - `-c`：创建打包文件。
  - `-x`：解包文件。
  - `-z`：使用 gzip 压缩。
  - `-v`：显示详细信息。
- **示例**：
  - `tar -czvf archive.tar.gz /home/user`

### 2. gzip
- **功能**：压缩文件。
- **示例**：
  - `gzip file.txt`

### 3. unzip
- **功能**：解压 ZIP 文件。
- **示例**：
  - `unzip archive.zip`

## 其他实用命令

### 1. history
- **功能**：显示命令历史记录。
- **示例**：
  - `history`

### 2. alias
- **功能**：创建命令别名。
- **示例**：
  - `alias ll='ls -la'`

### 3. wget
- **功能**：下载文件。
- **示例**：
  - `wget https://example.com/file.zip`

### 4. curl
- **功能**：传输数据（支持多种协议）。
- **示例**：
  - `curl -O https://example.com/file.zip`
