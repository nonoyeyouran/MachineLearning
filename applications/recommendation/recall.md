# 召回
## 召回算法
    召回算法可以分成三大类，分别是基于属性的召回、基于统计的召回和基于机器学习模型的召回
### 基于属性的召回
    包括热度召回、用户画像属性召回、用户人口学属性召回。这一层召回的粒度很粗，召回量很大。
### 基于统计的召回
    包括协同过滤（item-base、user-base、与模型结合）、矩阵分解
### 基于机器学习模型的召回
    包括I2I类、U2I类、U2U2I类等
#### I2I类
    这一类模型关键在于item embedding的学习，主要的方式是word2vec类和graphEmbedding的方法。
